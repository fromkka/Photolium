# Photolium

Photolium은 SillyTavern의 기본 채팅을 그대로 유지하면서, 버튼으로 들어갈 때만 감정 초상화와 문단별 타이핑 연출을 보여주는 선택형 대화 모드입니다.

## 사용법

1. 캐릭터 채팅을 엽니다.
2. 확장 설정의 **Photolium**에서 `happy.png`, `angry.png` 같은 감정 이미지를 추가합니다.
3. 필요하면 기본 배경 3종 또는 **사진 불러오기**로 배경을 고르고, 글자 크기·타이핑 속도·색상 테마를 조절합니다.
4. 입력창 왼쪽의 카메라 버튼 또는 **Photolium 열기**를 누릅니다.

포토리움 화면의 설정 카드에서 **코티지·데이라이트·로즈티·라벤더** UI 테마를 즉시 전환할 수 있습니다. 확장 설정에서는 기존 자유 색상 편집도 계속 사용할 수 있습니다.

Photolium 모드에서만 AI에게 현재 등록된 파일명 목록과 `[filename.png]` 형식을 지시합니다. 한 답변 안에서 문단마다 다른 태그를 사용할 수 있으며, 해당 문단이 시작될 때 초상화가 전환됩니다. 등록 이미지가 없거나 태그가 없으면 캐릭터의 SillyTavern 프로필 이미지를 사용합니다.

**초상화 페이드 전환**을 켜면 이전 표정 위로 다음 표정이 겹치며 크로스페이드됩니다. 포토리움에서 생성한 답변도 같은 SillyTavern 채팅에 저장되며, 기본 채팅으로 나갈 때 화면을 동기화합니다.

이미지는 데이터베이스가 아니라 SillyTavern의 사용자 이미지 폴더 아래 `photolium_*` 폴더에 저장됩니다. 확장 설정에는 경로와 테마 값만 저장됩니다.
